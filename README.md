## Design Choices

- **Typography**: Crimson Text (serif) for body text, Inter (sans-serif) for UI elements
- **Color Scheme**: Clean white background with subtle grays and blue accents
- **Layout**: Single-column, max-width 900px for optimal readability
- **Navigation**: Sticky header with smooth transitions

## Local Development

Simply open `index.html` in a web browser. No build process required.
